# torneio-volei
I Torneio RS/SC de Vôlei - Sistema de Sorteio e Gestão  Web App desenvolvido em Python com Streamlit para a gestão do I Torneio RS/SC de Vôlei (29/03/2026). O sistema permite o cadastro de equipes, exibição do regulamento oficial e a realização do sorteio de chaves (A e B) ao vivo com animações.  Organização: Cristiano Delfino.
